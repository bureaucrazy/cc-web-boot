3) Strings in Javascript
STRINGS in Javascript are IMMUTABLE! - cannot be changed.
So you need to create a new variable with the changes.
"" vs '' same in Ruby.
'' no escape character and no string interpolation.
"\world\" = "world"
'\world' = \world\
"world" = world

3 + "5" = 35
"3" = 5 = 35

NO INTERPOLATION in Javascript
need to use the + symbol.

"abc"[0] = "a"
"abc"[2] = "c"
"abc"[3] = undefined

parseInt("123") = 123
# NaN means Not a Number
parseInt("abc") = NaN
1 / 0 = Infinity

4) Variables
Always start with var
Always end with semi-colon;
Always use lowerCamelcase

# Var has local scope
# No var has global scope
var <var name>

var a = 1;
var b = 2;
var c = a + b;
# The next two lines will print out the value of 3
console.log(c); = 3

c
