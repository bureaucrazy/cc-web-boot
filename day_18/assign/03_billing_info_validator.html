Set up a form with the following fields:

  • First Name
  • Last Name
  • Address
  • City
  • Province
  • Postal Code
  • Phone (3 fields for (XXX)-XXX-XXXX)
  • Email
  • Credit Card number
  • Credit Card expiry (a field for month and a field for year)

Add validation to the form with the following rules when a submit button is pressed:

  • All fields are required
  • Use your Credit Card validator from previous assignments to make sure the card number is valid
  • Make sure the postal code is a valid Canadian postal code of the form (with or without a space)
  • Make sure the credit card expiry is in the future

If a field is breaking a rule, highlight it somehow with a CSS class and display the error underneath the field. Make sure to clear errors if the button is pressed again. Prevent the form from submitting unless validation passes.
